        ❌ Common Misconceptions
        • Partial factorisation: 6x² + 9x → 3(2x² + 3x) instead of 3x(2x + 3)
        • Difference of squares confusion: a² + b² is NOT factorisable over ℝ

        • Difference of squares pattern: a²−b² = (a+b)(a−b)

              👩‍🏫 Teacher Panel — Equation Solver
              ❌ Common Misconceptions
              • Not applying the same operation to both sides
              • Simultaneous: adding when they should subtract (or vice versa)
              • Quadratic formula: forgetting ± or using wrong sign under √
              • Thinking ax² + bx + c = 0 always has two solutions
              ✅ Teaching Strategies
              • Balance scale metaphor: whatever you do to one side, do to the other
              • Simultaneous: multiply to make coefficients equal, then eliminate
              • Quadratic formula: write out b²−4ac first (discriminant), then ±√
              • Always check solution by substituting back into original

                        👩‍🏫 Teacher Panel — Travel Graphs
                        ❌ Common Misconceptions
                        • Reading gradient as rise not rise/run — forgetting to divide by the time interval
                        • Flat section of d–t graph means "stopped" not "constant speed"
                        • Area under v–t graph gives distance, not just "area"
                        • Negative gradient on d–t = returning, not negative speed
                        ✅ Teaching Strategies
                        • Speed = gradient: always write gradient = Δy/Δx, with units km/h
                        • Split v–t graph into trapezoids/triangles to calculate area
                        • Sketch: label axes with units before drawing
                        • Connect to real journey: bus stop, station, turnaround

                              E2.8 — Direct & Inverse Proportion

                              y ∝ x · y ∝ x² · y ∝ 1/x · y ∝ √x · Find k, find missing values
                              0 correct
                              0 streak
                              0% score
                              👩‍🏫 Teacher Panel — Proportion
                              ❌ Common Misconceptions
                              • Direct proportion: assuming y ∝ x always means y = x (forgetting k)
                              • Inverse proportion: thinking y = −kx instead of y = k/x
                              • Not substituting to find k before using the formula
                              • y ∝ x² vs y ∝ √x — confusing which relationship applies
                              ✅ Teaching Strategies
                              • Always write y = kf(x) first, then substitute known values to find k
                              • Direct: as x increases, y increases (k > 0)
                              • Inverse: as x increases, y decreases; product xy = constant
                              • Verify with a second pair of values
                              📋 Proportion Types
                              StatementEquationGraph Shape

                                👩‍🏫 Teacher Panel — Differentiation
                                ❌ Common Misconceptions
                                • d/dx of a constant = the constant itself (should be 0)
                                • Power rule: bringing down the power but not reducing it by 1
                                • Thinking dy/dx = 0 always means maximum (need to check second derivative)
                                • Differentiating y = 3 → 3 instead of 0
                                ✅ Teaching Strategies
                                • Power rule: "multiply by power, reduce power by 1" — drill this mechanically first
                                • dy/dx = 0 → find x; then use d²y/dx² to classify
                                • d²y/dx² > 0 = minimum (smile shape); < 0 = maximum (frown)
                                • Connect to gradient: dy/dx at a point = gradient of tangent there

                                  Function notation · Domain & range · Inverse · Composite functions
                                  0 correct

                                  👩‍🏫 Teacher Panel — Functions
                                  ❌ Common Misconceptions
                                  • f⁻¹(x) means 1/f(x) — it's the inverse function, not the reciprocal
                                  • fg(x) means f(x) × g(x) — it's f(g(x)), composite
                                  • Applying operations in wrong order for composite functions
                                  • Domain restrictions: not checking for division by zero or √ of negative
                                  ✅ Teaching Strategies
                                  • Inverse: replace f(x) with y, swap x and y, rearrange for y
                                  • Composite fg(x): apply g first, then f to the result
                                  • Domain: list all x values that make the function undefined
                                  • Flowchart method for inverse — reverse the operations
