C3.1 — Cartesian Coordinates

👩‍🏫 Teacher Panel — Coordinates & Plotting
❌ Common Misconceptions
  • Reversing x and y: plotting (3, 5) as 3 up, 5 right
  • Negative coordinates: moving right for negative x
  • Assuming the origin is always visible in exam questions
  • Confusing quadrant numbers and sign combinations
✅ Teaching Strategies
  • "Along the corridor then up the stairs" — x first, then y
  • Colour-code axes: red x, blue y
  • Quadrant mnemonics: All Students Take Calculus (signs +/+, −/+, −/−, +/−)
  • Always label axes and scale before plotting
📝 Worked Example
Q: Plot the point (−3, 4) and state which quadrant it lies in.
  1. Start at the origin (0, 0).
  2. x = −3: move 3 units LEFT along the x-axis.
  3. y = 4: move 4 units UP from there.
  4. x is negative, y is positive → Quadrant II.

📐 Gradient

C3.2 — Gradient of a Line

Gradient = rise ÷ run = (y₂−y₁)/(x₂−x₁)
0 correct
0 streak
0% score
Difficulty: Core: integer gradients
👩‍🏫 Teacher Panel — Gradient
❌ Common Misconceptions
  • Swapping rise and run: computing Δx/Δy instead of Δy/Δx
  • Subtracting in opposite orders: (y₂−y₁)/(x₁−x₂)
  • Thinking horizontal lines have undefined gradient (they have 0)
  • Thinking vertical lines have gradient 0 (they are undefined)
  • Assuming gradient must be positive
✅ Teaching Strategies
  • Always draw a right-angled triangle between the two points on the line
  • Label which point is (x₁,y₁) and which is (x₂,y₂) before substituting
  • Check sign: line going up-right = positive; down-right = negative
  • Horizontal = flat = 0 gradient; Vertical = cliff = undefined
📝 Worked Example
Q: Find the gradient of the line through (1, 3) and (5, 11).
  1. Label: (x₁, y₁) = (1, 3) and (x₂, y₂) = (5, 11).
  2. m = (y₂ − y₁) ÷ (x₂ − x₁) = (11 − 3) ÷ (5 − 1).
  3. m = 8 ÷ 4 = 2.
  4. Check: line goes up-right → positive gradient ✓.

      C3.3 — Midpoint & Length of a Line

      Midpoint = average of coordinates · Distance = Pythagoras
      0 correct
      0 streak
      0% score
      Difficulty: Core: integer coordinates
      👩‍🏫 Teacher Panel — Midpoint & Distance
      ❌ Common Misconceptions
      • Distance formula: forgetting to square-root at the end
      • Subtracting coordinates in wrong order for distance (doesn't matter — but confuses students)
      • Midpoint: adding without dividing by 2
      • Squaring negative differences: (−3)² = 9 not −9
      ✅ Teaching Strategies
      • Midpoint = "meet in the middle" — average each coordinate separately
      • Distance: draw the right-angled triangle, then Pythagoras
      • Check: midpoint should lie exactly between both points on a sketch
      📝 Worked Example
      Q: A = (2, −1), B = (8, 7). Find midpoint M and distance AB.
      1. Midpoint: M = ((2+8)/2 , (−1+7)/2) = (5, 3).
      2. Distance: AB = √((8−2)² + (7−(−1))²) = √(36 + 64) = √100 = 10.
      📜 Formulae Reference
      Midpoint of (x₁,y₁) and (x₂,y₂)
      M = ((x₁+x₂)/2 , (y₁+y₂)/2)
      Distance between (x₁,y₁) and (x₂,y₂)
      d = √((x₂−x₁)² + (y₂−y₁)²)

          📏 Equation of a Line

          C3.4 — Equation of a Straight Line

          y = mx + c · gradient-intercept form · finding the equation
          0 correct
          0 streak
          0% score
          Difficulty: Core: integer m and c
          👩‍🏫 Teacher Panel — Equation of a Line
          ❌ Common Misconceptions
          • Confusing m (gradient) and c (y-intercept) in y = mx + c
          • Finding c by substituting x instead of y = 0
          • Writing the equation without simplifying (leaving fractions in m or c)
          • Thinking y = 3 has gradient 3 (it has gradient 0)
          ✅ Teaching Strategies
          • Always find m first, then substitute one point to find c
          • Check: substitute both given points into the equation to verify
          • Sketch: does the line look right? Positive gradient = up-right
          • Rearrange ax + by = c to y = mx + c form to read off m and c
          📝 Worked Example
          Q: Find the equation of the line through (2, 5) and (4, 11).
          1. Gradient: m = (11−5)/(4−2) = 6/2 = 3.
          2. Use y = mx + c → 5 = 3(2) + c → c = 5 − 6 = −1.
          3. Equation: y = 3x − 1.
          4. Check: when x=4, y = 12−1 = 11 ✓.

              ⊕ Parallel & Perpendicular Lines

              C3.5 / E3.5 — Parallel & Perpendicular Lines

              Parallel: same gradient · Perpendicular: gradients multiply to −1
              0 correct
              0 streak
              0% score
              👩‍🏫 Teacher Panel — Parallel & Perpendicular
              ❌ Common Misconceptions
              • Perpendicular gradient: negating instead of taking negative reciprocal (−m vs −1/m)
              • Thinking parallel lines must have the same y-intercept
              • Not checking: m₁ × m₂ = −1 for perpendicular
              • Horizontal and vertical lines: forgetting undefined/zero relationship
              ✅ Teaching Strategies
              • Parallel = "same slope, different intercept" — easy memory
              • Perpendicular: "flip and negate" — reciprocal and change sign
              • Verify with product: m₁ × m₂ must equal −1
              • Draw both lines on a grid — do they look right-angled?
              📝 Worked Example
              Q: Find the equation of the line perpendicular to y = 4x − 1 passing through (8, 3).
              1. Original gradient: m = 4.
              2. Perpendicular gradient: m⊥ = −1/4.
              3. Use y = −¼x + c → 3 = −2 + c → c = 5.
              4. Equation: y = −¼x + 5.
              📜 Key Rules
              Parallel Lines
              Two lines are parallel if and only if they have the same gradient.
              e.g. y = 3x + 1 ∥ y = 3x − 4 (both m = 3)
              Perpendicular Lines
              Two lines are perpendicular if m₁ × m₂ = −1.
              Perpendicular gradient = −1/m (negative reciprocal).
              e.g. m = 2 → perp. m = −½
              🔍 Classify the Relationship
              Are the two lines parallel, perpendicular, or neither?

                E3.6 — Intersection of Lines & Curves

                Solve simultaneously · Interpret intersections graphically
                0 correct
                0 streak
                0% score
                👩‍🏫 Teacher Panel — Intersections
                ❌ Common Misconceptions
                • Setting y = y but forgetting to solve the resulting equation
                • Parallel lines: students expect an intersection point (there isn't one)
                • Line meets curve: not forming the correct quadratic after substitution
                • Reading intersection coordinates from graph inaccurately
                ✅ Teaching Strategies
                • Set equal: if y = f(x) and y = g(x), then f(x) = g(x) and solve
                • Draw both graphs first — number of intersections = number of solutions
                • Check solution by substituting into both original equations
                📝 Worked Example
                Q: Find the intersection of y = 2x + 1 and y = −x + 7.
                1. Set equal: 2x + 1 = −x + 7.
                2. 3x = 6 → x = 2.
                3. y = 2(2) + 1 = 5.
                4. Intersection: (2, 5). Check in both equations ✓.

                  E3.7 — Loci & Regions in 2D

                  Describe loci · Identify regions satisfying inequalities
                  0 correct
                  0 streak
                  0% score
                  👩‍🏫 Teacher Panel — Loci & Regions
                  ❌ Common Misconceptions
                  • Locus of points equidistant from two points: drawing a circle instead of a perpendicular bisector
                  • Confusing "inside" and "outside" a circle region
                  • Region shading: shading the wrong side of the boundary line
                  • Forgetting dashed line for strict inequalities (<, >) vs solid for ≤, ≥
                  ✅ Teaching Strategies
                  • Loci: always start from the definition — "set of all points satisfying a condition"
                  • Test point (0,0): substitute into inequality to determine which side to shade
                  • Draw boundary first (solid or dashed), then shade the correct region
                  📝 Worked Example
                  Q: Shade the region where y > 2x − 3.
                  1. Draw the boundary line y = 2x − 3 as a dashed line (strict inequality).
                  2. Test a point: (0, 0) → 0 > −3 ✓ → the origin is in the region.
                  3. Shade the side of the line containing (0, 0): above the line.
                  📖 Loci Reference
                  ConditionLocus
                  Fixed distance r from a point PCircle, centre P, radius r
                  Equidistant from two points A and BPerpendicular bisector of AB
                  Equidistant from two linesAngle bisector of the two lines
                  Fixed distance from a lineTwo parallel lines, one each side
